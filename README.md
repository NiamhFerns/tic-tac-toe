# ❌ Tic Tac Toe ⭕

Cheap and cheerful implementation of Tic Tac Toe using Vue3 + Typescript.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
